## 🚀 Installation

```bash
$ git clone https://github.com/GuidoFavara/Tinder-Front
```


### 💻 Web

```bash
$ cd Tinder-Front
$ npm install
$ npm start
```

The application will pop-up in the browser on http://localhost:3000
